# He-thong-quan-ly-va-theo-doi-tien-do-cong-viec-ca-nhan
# Hệ thống quản lý và theo dõi tiến độ công việc cá nhân

Dự án xây dựng ứng dụng hỗ trợ quản lý, sắp xếp và theo dõi tiến độ công việc cá nhân hiệu quả, tích hợp cả giao diện người dùng (Frontend) và hệ thống xử lý dữ liệu (Backend).

##  Tính năng chính
* Đăng ký, đăng nhập tài khoản người dùng bảo mật.
* Thêm, sửa, xóa công việc cá nhân.
* Đặt thời hạn (Deadline) và mức độ ưu tiên (Cao, Trung bình, Thấp) kèm icon trực quan.
* Bộ lọc và tìm kiếm công việc thông minh theo tên và mức độ ưu tiên.

##  Công nghệ sử dụng
* **Frontend:** HTML, CSS, JavaScript (giao diện thân thiện, cute mượt mà).
* **Backend:** Node.js, Express.js.
* **Database:** SQL Server (Kết nối qua Windows Authentication).
